Project Overview

This repository contains an in-depth, interactive data analysis report focusing on the illustrious cricket career of Virat Kohli. Built using Power BI, the dashboard transforms raw match and innings data into compelling visuals, providing an analytical perspective on his performance across different formats, conditions, and opponents.

The attached file, Virat Kohli Stats report. Pbix, is the complete, interactive report and underlying data model.

How to View the Dashboard

To interact with the full dashboard, you will need Power BI Desktop installed on your machine.

1.	Install Power BI Desktop: If you do not have it, download and install the free application from the official Microsoft website.
2.	Download the File: Download the Amazon prime Dashboard.pbix file from this repository.
3.	Open the Report: Double-click the. pbix file or open it directly within Power BI Desktop.
4.	Explore: Use the slicers and filters within the report to interact with the data and uncover your own insights.
